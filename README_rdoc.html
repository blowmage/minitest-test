    minitest-test¶ ↑

    Minitest::Test for MiniTest 4

    DESCRIPTION:¶ ↑

    Enables Minitest 5 test syntax in MiniTest 4.

    Do you love the syntax changes made in Minitest 5 but are forced to stay on MiniTest 4? No problem. This small library will enable much of the Minitest 5 test syntax for you. Simply require "minitest/test" in your helper or your test file and you are good to go.

    require "minitest/test"
require "minitest/autorun"

class TestFoo < Minitest::Test
  def test_foo
    assert_equal "foo", Foo.new.foo
  end
end

    Best of all, this is forwards compatible with Minitest 5.

    INSTALL:¶ ↑

    Install with rubygems.

    gem install minitest-test
